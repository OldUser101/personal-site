# Uses

This is a list of stuff I use, that you might be interested in.
I "try" to keep this up-to-date.

## Software

I mainly use Linux-based systems, but occassionally BSD as well. I'm currently
using [NixOS](https://nixos.org), since I've found it gives me the greatest
control over my system. I'm also a maintainer of a couple of packages in
[Nixpkgs](https://github.com/NixOS/nixpkgs).

Other things I use, and highly recommend:

- [Kakoune](https://kakoune.org) for code editing, Vim if unavailable
- [Sway](https://swaywm.org/) window manager, but forked (I'm working
    on a few features of my own)
- [alacritty](https://alacritty.org) terminal emulator

## Hardware

I use some *weird* pieces of hardware, or at least some unexpected ones.

- Acer Aspire 3 Spin 14 laptop - the only computer I really use, very much
    not designed for some of the stuff I do, but it's (relatively) cheap.

- Microsoft IntelliMouse Explorer 3.0 - certainly my favourite mouse, I
    wouldn't really consider using anything else. Shame it was made ~20
    years ago now.

- Sony WH-CH700N headphones - got them a couple of years back for the
    bargain price of £25. Despite being made back in 2018, they aren't
    too bad for today (in my opinion).

- Various Apple iPods, all flash-modded and repaired, some using
    [Rockbox](https://www.rockbox.org).

- A jailbroken Amazon Kindle Paperwhite 3, that
[I ran Vim on](https://bsky.app/profile/ngill.net/post/3m6two7n4v222).

- Sony Walkman D-E220 portable CD player, probably one of the best value
    purchases I've ever made, for just £12.
